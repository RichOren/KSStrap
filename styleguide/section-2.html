2 Tiles

Tiles are used to give the user simple actions. Tiles should be limited to 2 or less actions on the tile.

2.1 Domain Tiles

Tiles can be specified by domain and be colored accordingly. By adding .tile-action as a wrapper div the entire tile becomes clickable.

2.2 Action Tiles

Action Tiles perform simple actions.

I am an action tile
This tile provides some basic actions for the user
Go Stop

	<div class="col-sm-6">
		<div class="tile-wrapper">
			<div class="tile tile-default">
				<div class="tile-heading">
					I am an action tile
				</div>
				<div class="tile-body">
					<div class="row">
						<div class="col-sm-12">
							<div class="h4 text-left">This tile provides some basic actions for the user</div>

					</div>
				</div>
